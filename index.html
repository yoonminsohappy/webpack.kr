번들해 보세요

설정 없이도 번들할 수 있고 사용자 지정 webpack.config.js를 제공할 수도 있어요.

const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    path: path.resolve(__dirname, 'dist'),
    filename: 'bundle.js',
  },
};

page.html

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    ...
  </head>
  <body>
    ...
    <script src="dist/bundle.js"></script>
  </body>
</html>

그다음 커멘드 라인에서 webpack 명령어를 실행해 bundle.js를 생성하세요.
